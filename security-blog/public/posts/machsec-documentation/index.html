What is machsec?

Machsec, is a custom tool I wrote to identify security mitigations on MachO binaries on iOS/macOS. The idea came to mind when i realized there is no “checksec” equivalent for these platforms, not one that can detect the unique mitigations that are present on these operating systems. Since the mitigations are super unique, and dont exist in any other operating system. Writing a tool from scratch was necessary. Detecting mitigations present in a binary are a crucial step in understanding what kind of exploits need to be found and abused in order to gain arbitrary code execution in them. For example, you can not abuse a dangling pointer if all of your pointers are signed. What are signed pointers? There is no such thing on other platforms? this is exactly the issue I tried to solve with this program.
